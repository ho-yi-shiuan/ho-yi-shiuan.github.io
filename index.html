Description:

For this graph, I used the building inventory file. This file has data of location, time built and size of each building. I visualized the relationship between total floors and square footage, I chose to mark data as points so that I can see if they are in a positive or negative relationship. I also added radio for user to select whether they want the points are colored by usage description or agency name.
